Набрать ниже указаннаю команду внутри папки chapter_02/openapi
    oapi-codegen --config=config.yml ./oapi.yaml
В результате создатся файл - openapi.gen.go

Установка protobuf tools:
    go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
    go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest


Установка go-money для chapter_03
https://github.com/Rhymond/go-money

Установка библиотеки для работы с postgres
go get github.com/jackc/pgx/v4

Установка библиотеки mongo-driver для работы с MongoDB
go get go.mongodb.org/mongo-driver/v2/mongo

Установка библиотеки stripe для работы
go get -u github.com/stripe/stripe-go/v83
